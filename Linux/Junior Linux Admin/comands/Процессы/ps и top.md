ps — snapshot процессов в момент выполнения

показать все процессы от всех пользователей:
ps aux
процессы конкретного пользователя:
ps -u
процессы по имени команды:
ps -C command
сортировка по памяти:
ps --sort=-%mem
сортировка по процессору:
ps --sort=-%cpu
Что означают колонки в ps aux:
· USER — владелец процесса
· PID — ID процесса
· %CPU — загрузка CPU
· %MEM — загрузка памяти
· VSZ — виртуальная память
· RSS — физическая память
· STAT — статус (R — running, S — sleeping, Z — zombie)
· COMMAND — команда

top — интерактивный просмотр в реальном времени

Горячие клавиши в top:
выход: q
сортировка по CPU: p
сортировка по памяти: M
убить процесс (ввести PID): k
изменить приоритет (renice): r
показать процессы пользователя: u
помощь: h
