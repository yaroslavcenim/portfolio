Корневой каталог: /

Основание системные каталоги:
1. /bin - основные программы командной строки (ls, cp, mv)
2. /sbin - основные системные исполняемые файлы
3. /etc -  файлы конфигурации системы и приложений
4. /lib - основные общие библиотеки
5. /boot - файлы необходимые для загрузки системы, включая ядро и файлы загрузчика.

Данные пользователей и приложения:
1. /home - личные каталоги пользователей
2. /root - каталог пользователя root (гарантирует пользователю вход в систему, даже если /home недоступен)
3. /opt - стороннее по
4. /usr - установление по и утилиты

Динамические и временные данные:
1. /var - переменные файлы (логи, кэши)
2. /tmp - временные файлы (часто удаляются после перезагрузи)
3. /run - информация о системе с момента последней загрузки (PID и др.)

Устройства и точки монтирования:
1. /dev - файлы устройств (жёсткие диски, терминалы и устройства ввода)
2. /media - точка монтирования съемных носителей (usb, sd)
3. /mnt - точка временного монтирования

Системная информация:
1. /proc - виртуальная файловая система (информация о запущенных процессах и параметрах ядра в реальном времени)
2. /srv - данные специфичные для сайта, обслуживаемых системой (файлы для веб сервера)
