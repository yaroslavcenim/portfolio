SELinux (Security-Enhanced Linux) — система обязательного контроля доступа (MAC).
Она работает поверх обычных прав Linux и может запрещать даже то, что разрешено chmod, если не совпадает контекст безопасности.

Три режима работы:
1. Enforsing - SELinux включён, блокирует нарушения
2. Permissive - SELinux включён, но не блокирует, только пишет в лог
3. Disabled - SELinux выключен полностью
Проверка режима: getenforce

Режим можно переключать в конфиге: /etc/selinux/config
В строке: SELINUX=(режим, с маленькой буквы)
Для применения настроек, перезагрузить систему
