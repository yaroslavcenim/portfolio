Обе команды помогают найти, где лежит программа.

which — показывает путь к исполняемому файлу
Ищет только в директориях, перечисленных в $PATH.
Показывает только первый найденный.

whereis — ищет шире

Показывает:
· бинарник (исполняемый файл)
· исходники (если есть)
· man-страницы

Ключи whereis:
Только бинарники: -b
Только исходники: -s
Только man: -m
