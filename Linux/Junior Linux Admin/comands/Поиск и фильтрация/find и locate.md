find — поиск файлов и папок в реальном времени

Основные команды find:
Поиск по имени: -name "имя"
Поиск по имени без учёта регистра: -iname "имя"
Тип объекта: -type (f — файл, d — папка, l — ссылка)
Размер:  -size (+100M, -1M, 10M)
Время изменения в днях: -mtime (-7, +30)
Время изменения в минутах  : -mmin
Владелец: -user
Группа: -group
Права доступа: -perm -"4755"
Удалить найденое: и-delete
Выполнить команду: -exec (подробнее в папке "Логи")
Выполнить команду с подтверждением: -ok "структура как и -exec"

locate (найти по базе) — мгновенный поиск по индексированной базе

Основные команды locate:
Обновление базы: updatedb
Без учёта регистра: -i
Показать только количество: -c
Ограничить вывод N строками: -l N
Только существующие файлы: -e (из за updatedb)

Поиск по регулярному выражению: -r "^/home" ("^"начало пути)
