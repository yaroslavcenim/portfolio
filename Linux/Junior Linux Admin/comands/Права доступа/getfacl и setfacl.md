ACL - Access Control Lists "Списки контроля доступа"

Ключи setfacl

-m — modify (добавить/изменить)
-x — xremove (удалить для конкретного)
-b — bremove all (удалить все)
-R — рекурсивно (для папок)
-d — default (для будущих файлов в папке) (используется обычно вместе с -m)

Основные команды:
getfacl "файл" посмотреть текущие ACL
setfacl -m u:имя:rwx "файл" добавить права для пользователя
setfacl -m g:группа:rx "файл" добавить права для группы
setfacl -x u:имя "файл" удалить ACL для пользователя
setfacl -b "файл" удалить все ACL (вернуться к обычным правам)
