1. Показать диски: lsblk
2. Выбрать диск для создания раздела: fdisk “путь к диску”
3. Создание нового раздела: n
4. Выбрать основой раздел: p или e для расширенного
5. Выбор номера раздела
6. Выбор сектора (просто минимальное значение)
7. Выбор размера: +10G, +100M, +1T
8. Изменяем тип раздела: t
9. Нажимаем L и ищем код для : Linux swap (мб 82)
10. Запись изменений: w
11. Проверка: partprobe “путь к диску”
12. Форматируем: mkswap “путь к разделу”
13. Постоянное монтирование раздела: редактирование файла /etc/fstab/ по структуре:(см. ниже)
14. Проверка ошибок: swapon -a
15. Лучше сделать: systemctl daemon-reload
