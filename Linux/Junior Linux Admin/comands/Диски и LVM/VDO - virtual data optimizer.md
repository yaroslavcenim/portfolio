1. Установка пакетов: lvm2 kmod-kvdo vdo
2. Создаем физ. томы
3. Создаем группу томов
4. lvcreate —type vdo —name «имя создаваемого тома» —size “физический размер” —virtualsize “виртуальный размер (в соотношении 1:10)” «имя группы»

 «—» это двойное тире

5. Форматирование тома: mkfs.”формат” -K “путь к созданному VDO”
6. Создаем точку монтирования: mkdir
7. Постоянное монтирование раздела: редактирование файла /etc/fstab/ по структуре:(см. ниже)
8. Проверка: mount -a

Редактирование файла /etc/fstab/ (между  заполняемыми данными отступать нажатием «Tab»)
1. Путь к разделу vdo
2. Путь к точке монтирования
3. Тип файловой системы 
4. defaults
5. 0
6. 0

Для мониторинга vdo: vdostats —human-readable
