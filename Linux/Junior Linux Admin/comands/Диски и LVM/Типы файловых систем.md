Просмотр файловых систем: df -Th
-T показать тип файловой системы
-h читаемый формат

ext4 - надежный и стандартный выбор для большинства сценариев использования

Btrfs (B-tree FS) - файловая система с расширенными функциями (snapshots)

XFS - высокопроизводительная файловая система с журналированием (работа с большими файлами)

NTFS и FAT - файловые системы Windows (полная продержка системой Linux)

HFS+ - файловая система macOS (поддержка только для чтения системой Linux)
