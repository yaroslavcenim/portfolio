Архивация - собрать много файлов в один (tar)
Сжатие - уменьшить размер файла (gzip, bzip2, xz)

tar — Tape ARchiver
-c create -- создать архив
-x extract -- распаковать
-t list -- посмотреть содержимое
-f file -- указать имя
-z сжать через gzip
-j сжать через bzip2
-v verbose -- показать процесс

gzip / gunzip - сжатие отдельного файла
Структура: gzip / gunzip "имя файла"

zip / unzip - сам сжимает
Структура: 
zip -r (для папок) "имя.zip" папка/
unzip "имя архива"

tar (--exclude) структура:
tar --exclude'имя файла или каталога(без /)' ... продолжение команды
