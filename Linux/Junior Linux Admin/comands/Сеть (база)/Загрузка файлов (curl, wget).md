curl и wget — скачивание файлов и работа с HTTP
wget — простой загрузчик

Команды wget:
Загрузка файла: wget "ссылка на файл"
Сохранить под другим именем: wget -O "новое имя" "ссылка на файл"
Рекурсивно скачать файл (весь сайт): wget -r "ссылка на сайт"
Продолжить скачку wget -c "ссылка на файл"

curl — мощный инструмент для работы с URL

Команды curl:
Вывести страницу в терминал: curl "ссылка"
Сохранить как файл: curl -O "ссылка"
Сохранить под другим именем: curl -o "name" "ссылка"
Только заголовки HTTP: curl -l "ссылка"
Отправить POST-запрос: curl -X POST -d "name=test" "ссылка"
Авторизация: curl -u user:pass "ссылка"
Игнорировать SSL-ошибки: curl -k "ссылка"
