ss -tulpn — просмотр открытых портов

ss (socket statistics) — современная замена netstat. Показывает информацию о сетевых соединениях, открытых портах, сокетах.

Основные ключи:
TCP-соединения: -t
UDP-соединения: -u
Только слушающие (listening) порты: -l
Показать процесс использующий порт: -p
Показывать IP и порты числами: -n
Все соединения: -a
