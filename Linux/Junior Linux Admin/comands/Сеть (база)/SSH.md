Копирование файлов по ssh:
1. Копирование с сервера на локальную машину: scp пользователь@ip-адрес:/путь/к/файлу /локальная/папка
2. Копирование с локальной на сервер: scp /локальный/файл пользователь@ip-адрес:/путь/на/сервере/
3. Для копирования папки ключ: -r

Создание ключей:
1. ssh-keygen -t rsa -b 4096
Будут созданы 2 файла:
   ~/.ssh/id_rsa (приватный)
   ~/.ssh/id_rsa.pub (публичный)

Копирование ключей:
ssh-copy-id пользователь@ip-адрес

3.5. Важные файлы и права

· Приватный ключ (id_rsa) должен иметь права 600 (только чтение для владельца)
· Публичный ключ (id_rsa.pub) добавляется в ~/.ssh/authorized_keys на сервере
· Папка ~/.ssh на сервере должна иметь права 700
