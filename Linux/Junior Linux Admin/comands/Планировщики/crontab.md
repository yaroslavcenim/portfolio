crontab (cron table) - используется для управления расписанием автоматиких задач

Основные команды:
Открыть файл конфигурации для текущего пользователя: -e
Список всех запланированных задач: -l
Удалить файл задач текущего пользователя: -r
Редактировать задачи конкретного пользователя: -u "имя пользователя" -e
