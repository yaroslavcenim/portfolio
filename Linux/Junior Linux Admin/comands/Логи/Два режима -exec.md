
1. С \; — для каждого файла отдельно

```bash
find . -name "*.log" -exec cp {} /backup/ \;

cp запускается отдельно для каждого файла. Если файлов 1000 — команда выполнится 1000 раз.
```


2. С + — все файлы одной пачкой

```bash
find . -name "*.log" -exec cp {} /backup/ \+
```

Все найденные файлы передадутся одной командой:

```bash
cp file1.log file2.log file3.log /backup/
```

Так быстрее, но работает не со всеми командами (должны принимать много аргументов).
