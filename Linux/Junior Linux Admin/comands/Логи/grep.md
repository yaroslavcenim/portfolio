Базовый поиск: grep "шаблон" файл
Рекурсивный поиск: -r
Игнорирование регистра: -i
Инверсия поиска: -v
Поиск целого слова: -w
Номера строк: -n
Только имена файлов: -rl
Вывод контекста: -A (после) -B (до) -C (до и после)

Работа с регулярными выражениями (grep -E)
Или: grep -E "слово1|слово2"
Начало строки: "^ шаблон"
Конец строки: "шаблон$"
Повторения: grep -E "a{2,4}" - поиск от 2 до 4 повторений символа "a"

Примеры:
ps aux | grep "nginx"
grep -r "ошибка" /var/log/
