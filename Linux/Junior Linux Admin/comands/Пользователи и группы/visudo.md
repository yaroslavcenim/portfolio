visudo — специальная команда для редактирования /etc/sudoers (проверяет синтаксис, чтобы не сломать sudo)

Синтаксис строки в sudoers:
  пользователь  хост=(от_кого)  команды

Пример:
  john    ALL=(ALL)       ALL

Это значит: пользователь john может с любого хоста (ALL) выполнять любые команды (ALL) от любого пользователя (ALL).

Права для группы:
  %group   ALL=(ALL)       ALL

Без пароля:
  john    ALL=(ALL)       NOPASSWD: ALL

Ограничение по командам:
john    ALL=(ALL)       /usr/bin/systemctl, /usr/bin/apt
