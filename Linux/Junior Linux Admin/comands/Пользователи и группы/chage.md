chage (change age) — команда для управления сроком действия пароля пользователя.

Основные команды chage:
Просмотр информации о старении пароля: 
-l "username"

Максимальное число дней действия пароля: 
-M "days" "username"

Минимальное число дней между сменами: 
-m "days" "username"

Предупреждение за N дней до истечения:
-W "days" "username"

Дата окончания учётной записи (YYYY-MM-DD): 
-E 2025-12-31 "username" (-1 или never)

Последнее изменение пароля (0 = сброс):
-d 0 username
