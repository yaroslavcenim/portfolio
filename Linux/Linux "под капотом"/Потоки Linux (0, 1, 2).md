В Linux есть три стандартных потока:

· 0 — stdin (ввод, обычно клавиатура)
· 1 — stdout (вывод, обычно экран) — сюда попадают нормальные результаты
· 2 — stderr (ошибки, тоже экран) — сюда пишутся сообщения об ошибках

Часто применяется:
2>/dev/null убрает ошибки из вывода
2>file перенаправляет ошибки в файл file
&>file или ... > file 2>&1 перенаправляют и вывод и ошибки
<file позволяет команде брать данные из файла
