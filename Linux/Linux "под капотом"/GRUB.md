Основная цель GRUB - загрузить ядро, но ему нужны инструкции о том как и где его найти. 

Редактирование параметров ядра в меню GRUB: нажать 'e'
Параметры:
1. initrd - указывает расположение начального RAM- диска, временной корневой файловой системы, загружаемой в память.
2. BOOT_IMAGE - определяет путь к файлу образа ядра, который должен быть загружен.
3. root - указывает на расположение фактической корневой файловой системы. Ядро использует этот путь для поиска процесса init.
4. ro - стандартный параметр, который предписывает ядру изначально монтировать корневую файловую систему в режиме только для чтения (это позволяет запустить проверку файловой системы до внесения изменений).
5. quiet - подавляет большинство подробных сообщений о загрузке, обеспечивая менее многословный экран запуска.
6. splash - включает отображение графической заставки во время процесса загрузки вместо текстовых сообщений.

Конфигурация GRUB
  Основной конфиг: /etc/default/grub
  Для того чтобы изменения вступили в силу: 
  update-grub
  grub2-mkconfig -o /boot/grub2/grub.cfg

Обозначения параметров:
    GRUB_DEFAULT=0 - загружается первая запись в меню GRUB.
    GRUB_TIMEOUT=5 - пауза в секундах перед автозагрузкой.
    GRUB_DISTRIBUTOR="имя" - название дистрибутива в меню GRUB.
    GRUB_CMDLINE_LINUX_DEFAULT="quiet splash" - параметры, которые передаются ядру.
    GRUB_CMDLINE_LINUX - доп параметры, которые работают во всех режимах.
